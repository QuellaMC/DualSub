# 功能特性

## 核心功能

- 双语字幕显示：同时显示原文和翻译
- 多平台支持：适配 Netflix 与 Disney+
- 多翻译服务商：Google、Microsoft、DeepL、OpenAI 兼容
- 智能翻译：自动回退与智能批处理

## 自定义选项

- 灵活布局：上下或左右排列
- 外观控制：字体大小、间距和显示顺序
- 垂直定位：精确控制字幕位置
- 时间偏移：微调字幕同步
- 多语言界面：支持 EN、ES、JA、KO、ZH-CN、ZH-TW

## 高级功能

- AI 上下文分析：文化、历史、语言解读（OpenAI 或 Google Gemini）
- 性能调优：批处理大小与请求延迟
- 高级选项：服务商特定设置与全面配置
- 日志系统：可配置日志级别
- 官方字幕集成：优先使用平台原生字幕（Netflix）

参见：`docs/zh/platforms.md` 与 `docs/zh/providers.md`。
